mirror-toolkit

Select working mirrors and generate environment values for reproducible builds.

Install

curl -fsSL https://example.invalid/setup-mirrors.sh -o setup-mirrors.sh

Usage

./setup-mirrors.sh --config mirrors.json --output .env.mirrors --report mirror-report.json

Generated env vars

Docker, pip, npm/yarn/pnpm, Maven, Ubuntu APT, Debian APT mirror variables are generated.
