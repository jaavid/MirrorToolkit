mirror-toolkit

Select working mirrors and generate environment values for reproducible builds.

Dockerfile optimizer

Markers:

  • # mirror-toolkit: enable-apt-rewrite (opt-in apt rewrite block; affects image build only)
  • # mirror-toolkit: enable-maven-mirror (opt-in active Maven settings.xml block)

Warnings: APT rewrite only happens with marker; Maven active config only happens with marker; no file is uploaded to any server, processing stays in your browser.
