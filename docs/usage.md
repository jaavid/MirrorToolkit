# Usage

```bash
./setup-mirrors.sh [mirrors.json]
./setup-mirrors.sh --config mirrors.json --output .env.mirrors --report mirror-report.json --timeout 6 --no-apply
./setup-mirrors.sh --help
```

Options:
- `--config mirrors.json`
- `--output .env.mirrors`
- `--report mirror-report.json`
- `--timeout 6`
- `--apply`
- `--no-apply`
- `--restart-docker`
- `-h, --help`
